## マクロ

JavaScript (JScript?) で書いて `Macros` ディレクトリに置けば使える。

### add_headSpace

- 選択範囲の行頭に全角空白を追加する
- ただし黒四角 `■`が行頭にある場合は追加しない

### date_string

- 直近の日付と曜日の文字列を生成する
- 日程調整のやり取りの際によく使っている
